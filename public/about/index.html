Project Scope

MH Wilds Skill-First Gear Optimizer is a skill-first planner for Monster Hunter Wilds. It helps you define target skills, search valid armor/deco combinations, and evaluate charm requirements for each result.

Tools In This Site

Skill-First Gear Optimizer: Build search tool for target skills, armor filters, decoration pool control, near-miss handling, and charm-mode evaluation.

Owned Charm Builder: Form-based page for creating/editing owned charms, with JSON import/export and direct sync to optimizer storage for Owned Charm mode.

Data Source

Build data comes from MHDB Wilds API endpoints for armor, armor sets, skills, decorations, and charms. The app fetches and caches data client-side.

Ranking Model

Valid builds are ranked by score metrics such as defense, slot utility, and target-skill efficiency. Charm suggestion mode can additionally re-rank using best post-build charm completion.

Beginner Filter Setups

Starter Realistic: Charm Mode = Off, Allow alpha/gamma = On, All decorations selected, Near-miss disabled.

Starter With Guidance: Charm Mode = Suggest After, Near-miss enabled, Near-miss max missing points = 2, Show meets without charm = On, Show meets with best charm = On.

Advanced Filter Setups

RNG Theorycraft: Charm Mode = Suggest After, Near-miss enabled, Near-miss max missing points = 4-6, Hide HIGH charm dependence = Off.

Owned Charm Validation: Charm Mode = Owned, import your charm library, Near-miss enabled (2-3), Show meets without charm = On and Show meets with best charm = On to compare realistic vs charm-assisted outputs.
