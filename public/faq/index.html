FAQ

What does this optimizer calculate?

It searches armor, charm, and decoration combinations that meet selected skill targets and ranks valid results by defense and slot efficiency.

Where does the gear data come from?

The app uses MHDB Wilds API data for armor, armor sets, skills, decorations, and charms.

Why are some builds missing?

Filters such as alpha or gamma armor toggles, selected decorations, and requested skill levels can remove potential builds from the search space.

Does this tool calculate weapon damage?

No. This project is skill-first build planning for armor sets and related setup.
